# VAR_MIRI_Assignment_2


## Desarrollo:

- Instalar node
- La instalación debería dejar disponible NPM 
- En una terminal ejecutar `npm install` para instalar las dependencias (definidas en el package-lock.json o package.json)
- Ejecutar en una terminal `npm run dev` para levantar el servidor de vite (se hace con este comando porque ya está configurado correctamente en el package.json )


## Build:
- Ejecutar en una terminal `npm run build` para generar la versión de despliegue en el directorio /docs (configurado asi para facilitar trabajo con github pages)

## Despliegue
- Github pages está configurado para actualizar la versión del proyecto según los cambios que se hagan en la rama master en el directorio /docs
